## Usage
<code>

</code>

## additional commands

* add reference to th eleft lower corner

<code latex>
\addcite{reference_name}
</code>

* enable dualview mode
* the generated pdf has the double width, the right one is for notes
* to display this on linux, I'm using "dspdfviewer".

<code latex>
\setDualView
</code>
